[[overview-http-verbs]]
== HTTP методы

RESTful notes tries to adhere as closely as possible to standard HTTP and REST conventions in its
use of HTTP verbs.

В данном REST API используются стандартные HTTP методы.

|===
| Метод | Описание

| `GET`
| Используется для получения объекта

| `POST`
| Используется для создания объекта

| `PUT`
| Используется для существующего объекта, включая частичное обновление

| `DELETE`
| Используется для удаления существующего объекта
|===

[[overview-http-status-codes]]
== HTTP коды статусов

В данном REST API используются стандартные коды статусов ответов HTTP.

|===
| Код статуса | Описание

| `200 OK`
| Запрос выполнен

| `201 Created`
| Новый объект успешно создан

| `500 Internal server error`
| Внутренняя ошибка сервера
|===

[[overview-hypermedia]]
== Hypermedia

RESTful Notes uses hypermedia and resources include links to other resources in their
responses. Responses are in http://stateless.co/hal_specification.html[Hypertext Application
Language (HAL)] format. Links can be found beneath the `_links` key. Users of the API should
not create URIs themselves, instead they should use the above-described links to navigate
